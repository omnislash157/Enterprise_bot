# Available Skills

Use `/skill <name>` to load a skill. Skills provide specialized capabilities.

| Skill | Description | Trigger |
|-------|-------------|---------|
| `db` | PostgreSQL queries, schema inspection, CSV export | database, sql, tables, export |
| `etl` | ETL pipeline generation, data transforms | etl, pipeline, transform |
| `excel` | Excel/XLSX export with formatting | excel, xlsx, spreadsheet |
| `powerbi` | Power BI dataset push, DAX generation | powerbi, dashboard, report |
| `schema` | Schema design, migrations, star schema | schema, migration, ddl |
| `profile` | Data profiling, quality checks, stats | profile, quality, nulls |

When user mentions trigger words, suggest loading the relevant skill.
