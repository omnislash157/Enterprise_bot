PURCHASE ORDER ENTRY
================================================================================

Path: 9-Purchasing > 1-Purchase Order Entry

1. Enter operator number (first order of session only)
2. Enter vendor number or "A" to search vendors
3. Pick Up order? Y/N
   - If Y: Select M (Managed) or B (Backhaul)
   - If Managed: Enter pickup date, then arrival date
4. Enter PO date (format: MMDDYY, e.g., 011924 for Jan 19, 2024)
5. Use left/right arrows to select ETA based on lead time
6. Dropship defaults to N
7. Warehouse defaults to 01 (Wayne)
   - Press F4 to change: 02 (Amsterdam) or 05 (Stroudsburg)
8. Enter item numbers and quantities
9. Enter through until Print (Y/N) - enter Y
10. Select P/F/E (Print, Fax, Email) - enter E
11. Confirm email address and CC
12. Add message if needed
13. Select 1=Send to transmit order

NEW ITEM SETUP
================================================================================

APPROVAL REQUIRED: All new items must be approved by a manager. Sales requests 
require DSM approval.

Required Specifications:
- Pack size
- Net weight & Gross weight
- Case cube
- Manufacturer number
- Item number
- UPC
- TI/HI
- Billing type
- Shelf life & Guaranteed shelf life

Setup Procedure:
----------------
1. Search for similar items (keep item numbers grouped)
2. Select similar item number, note it
3. Return to empty item screen
4. Press F1, enter the reference number
5. Select from populated numbers
6. Enter A to begin

Key Fields:
-----------
Field 2 - Item Description:
  Format: Category first, then specifics
  Example: "CHIP POTATO BBQ" (not "BBQ Potato Chip")
  Example: "DRESSING ITALIAN HOMESTYLE" (not "Homestyle Italian Dressing")

Field 6 - Billing Type:
  1 = By case
  2 = By weight (LB, OZ)
  3 = Catchweight (variable weight per delivery)

Field 8 - Pieces Per Case:
  Enter split quantity, or 1 if no split
  Note: Avoid splits except for sturdy self-contained items (gallon jugs, etc.)

Field 10/11 - Net and Gross Weight
Field 14 - Case Cube
Field 17 - Landed Cost
Field 20 - Unit type (bag, box, bottle) if applicable
Field 21 - Manufacturer and item number
Field 22 - Temperature zone
Field 27 - TI/HI
Field 28 - UPC
Field 32 - Category number (enter ? then spacebar to search)
Field 38 - Vendor number
Field 40 - Alpha name (search term for sales reps)
Field 41 - Brand name

Lockout Options (Field 43):
---------------------------
X = Cannot key out until back in stock or on order (ETA-based release)
Y = Locked to everyone unless inventory on hand
S = Locked to BOE only
D = Locked to street sales only

Other Flags:
------------
Field 45 - Taxable: Y for non-food items only
Field 46 - DWO (Done When Out): Item discontinued
Field 47 - Special Order: Y if one-time purchase, N if stocked
Field 53 - Exclude Special: Y to keep special order flag permanent

CLOSE CODING AN ITEM
================================================================================

Path: 5-Inventory Control > 1-Inventory File Maintenance

1. Enter item number
2. Left arrow to movement screen

4. Enter salesman number OR 6-digit customer number

Warehouse Close Codes:
- C = Wayne
- P = Stroudsburg

Note: Close coding restricts item to specific salesman/customer. Different from 
lockout (Field 43) which blocks all sales.

================================================================================

================================================================================
VENDOR ALLOWANCES & CHARGES
================================================================================

Path: 9-Purchasing > 28-Vendor Master File Maintenance > AL (Allowance Maintenance)

Header Setup (Select H):
------------------------
- Affect Totals: Y (deduct from invoice) or N (no effect)
- COG: Y (deduct from cost of goods) or N (no effect)
- Total PO: 0 for item-level allowances, or negative amount for PO-wide

Item Allowances (Select C):
---------------------------
1. Enter 6-digit item number
2. Select: 1=Allowance or 3=Charge
3. Select basis: Case price, Net weight, Catch weight, Percentage, or Gross weight
4. Select: 2=Add allowance value or 4=Add charge value
5. Up arrow, then Y to save

Note: Allowances reduce COG. Charges add to COG.
To apply to open PO: Enter the PO and tab through items.

FREIGHT FIELDS
================================================================================

Freight Allowance 1: Pickup allowance - nets down on invoice
Freight Allowance 2: Pickup allowance - NO invoice effect (for billbacks)
Freight Charge 1: Delivery freight - affects PO (delivery orders only)
Freight Charge 2: Delivery freight - NO invoice effect

IMPORTANT: Review PO freight screen before confirming. Contact manager if unsure.

Minimum backhaul freight: $200
